# Banco de Dados

Esta pasta contém todos os dados persistentes, configurações e o código fonte do bot.

## Estrutura
- `fonte/`: Código principal em JavaScript.
- `configuracao/`: Arquivos de configuração (JSON, etc).
- `estado/`: Logs e estados temporários.
- `sessao/`: Tokens de autenticação do WhatsApp.
- `testes/`: Testes automatizados.
- `scripts/`: Scripts de manutenção e utilitários de linha de comando.
