# Código Fonte (fonte)

Contém a lógica de negócio principal do bot.

## Estrutura
- `conectar.js`: Ponto de entrada para conexão com Baileys/WhatsApp.
- `robo.js`: Processador central de mensagens e comandos.
- `contexto.js`: Singleton com o contexto global da aplicação.
- `servicos/`: Módulos de serviço especializados.
- `utilitarios/`: Funções auxiliares.
- `aplicativo/`: Controladores e fluxos de alto nível.
- `menus/`: Definições dos menus de resposta.
