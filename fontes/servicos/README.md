# Serviços

Módulos responsáveis por tarefas específicas do bot, como IA, logs e limpeza.

## Exemplos
- `cliente_groq.js`: Integração com a API Groq.
- `limpador_sessao.js`: Manutenção da pasta de sessões.
- `gerenciador_lock.js`: Controle do arquivo package-lock.json.
- `seguranca.js`: Filtros de segurança e auditoria.
